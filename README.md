# autodice
autodice transfer bot gamble dice 

## Clone project
`git clone https://github.com/raserrano/autodice.git`

## Installation process
`cd autodice`
`npm install`

## Execution
`node index.js`

## Configuration
Create a .env file with the following settings
```
ACCOUNT=account
TOKEN=token
LOW='under 75 '
HIGH='over 6 '
TO=magicdice
WIF=wif
AMOUNT='0.100 STEEM'
FREQ=20
```

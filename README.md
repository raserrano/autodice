# autodice
autodice transfer bot gamble dice 

## Clone project
`git clone https://github.com/raserrano/autodice.git`

## Installation process
`cd autodice`
`npm install`

## Execution
`node index.js`

## Configuration
Create a .env file with the following settings
```
ACCOUNT=youraccount
MEMO=memo
TO=target
WIF=
AMOUNT=0.100 STEEM
```
